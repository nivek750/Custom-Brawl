Welcome to the Custom-Brawl lair.

This game about maximum customisability and is available on android here: https://play.google.com/store/apps/details?id=softtoasty.custombrawl

It is a 2d fighting game made for the modder! Users can store their created characters onto the sd card and load them onto the game.

Any help is welcome!
